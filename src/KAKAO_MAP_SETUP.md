# 🗺️ 카카오맵 API 설정 가이드

짐프리 앱에서 실제 카카오맵을 사용하기 위한 설정 방법입니다.

## 📋 API 키 발급 방법

### 1. Kakao Developers 가입 및 앱 등록

1. [Kakao Developers](https://developers.kakao.com/) 접속
2. 우측 상단 "로그인" 클릭 (카카오 계정으로 로그인)
3. 상단 메뉴에서 **내 애플리케이션** 클릭
4. **애플리케이션 추가하기** 버튼 클릭
5. 앱 이름 입력 (예: "짐프리") 후 저장

### 2. 플랫폼 등록

1. 생성한 앱을 클릭하여 상세 페이지 진입
2. 좌측 메뉴에서 **앱 설정 > 플랫폼** 클릭
3. **Web 플랫폼 등록** 클릭
4. 사이트 도메인 입력
   - 로컬 개발: `http://localhost:3000`
   - 배포 시: 실제 도메인 주소
5. **저장** 클릭

### 3. JavaScript 키 확인

1. 좌측 메뉴에서 **앱 설정 > 앱 키** 클릭
2. **JavaScript 키** 복사

## ⚙️ 코드에 API 키 적용

### `/components/KakaoMap.tsx` 파일 수정

33번째 줄의 `KAKAO_MAP_API_KEY` 상수에 발급받은 JavaScript 키를 입력하세요:

```tsx
// 수정 전
const KAKAO_MAP_API_KEY = ''; // 비어있으면 Mock 지도 사용

// 수정 후 (예시)
const KAKAO_MAP_API_KEY = '1234567890abcdef1234567890abcdef';
```

**참고:** API 키가 비어있으면 자동으로 Mock 지도가 표시되며, 에러 없이 정상 작동합니다.

## 🎯 적용된 페이지

카카오맵 API는 다음 페이지에서 사용됩니다:

- **짐 보관하기** (`/components/StorageMapPage.tsx`)
  - 보관 가능한 장소들을 지도에 마커로 표시
  - 남색(indigo) 계열 마커 사용

- **짐 배달하기** (`/components/DeliveryMapPage.tsx`)
  - 배달 가능한 가맹점들을 지도에 마커로 표시
  - 보라색(purple) 계열 마커 사용

## 🔄 Fallback 기능

API 키가 설정되지 않았거나 로드에 실패한 경우, 자동으로 **Mock 지도**가 표시됩니다:
- 가상의 도로 라인이 그려진 배경
- 현재 위치 표시 (빨간색 점)
- 클릭 가능한 마커들
- 화면 상단에 API 키 설정 안내 메시지 표시

## 📍 지도 좌표 정보

현재 설정된 위치는 **서울 강남역 일대**입니다:
- 지도 중심: 위도 `37.498095`, 경도 `127.027610`
- 각 가맹점의 실제 위도/경도 좌표가 Mock 데이터에 포함되어 있습니다

## 🔒 보안 주의사항

- JavaScript 키는 클라이언트 사이드에서 사용되므로 외부에 노출될 수 있습니다
- **도메인 제한 설정**을 통해 허용된 도메인에서만 API를 사용하도록 제한하세요
- 프로덕션 환경에서는 환경 변수를 사용하여 API 키를 관리하는 것을 권장합니다

## ❓ 문제 해결

### 지도가 표시되지 않는 경우
1. API 키가 올바르게 설정되었는지 확인
2. 도메인이 Kakao Developers에 등록되었는지 확인
3. 브라우저 콘솔에서 에러 메시지 확인

### Mock 지도가 계속 표시되는 경우
- API 키를 설정한 후 페이지를 새로고침하세요
- 브라우저 캐시를 삭제한 후 다시 시도하세요
